🚀 快速开始

在终端中运行以下命令启动本地服务器：

cd src
python3 -m http.server 8000

然后访问 http://localhost:8000

📱 Android开发

准备构建Android应用？运行以下命令：

npm install
npx cap add android
npx cap open android
